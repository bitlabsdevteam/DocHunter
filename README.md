# DocHunter

AI Agentic Healthcare Locator for Japan.

## Purpose
DocHunter is an AI agentic web application that helps users:
- describe medical symptoms
- understand urgency level guidance (non-diagnostic)
- locate nearby clinics and hospitals in Japan
- identify likely specialty matches
- move toward appointment booking
- receive confirmation by email

## Product Direction
This project is being designed as a lightweight, fast, multilingual healthcare locator agent with:
- English / Japanese support
- Perplexity-backed clinic discovery research
- secure onboarding with consent
- geolocation-aware provider matching
- agentic tool orchestration
- practical deployment paths using modern web infrastructure

## Current Status
Initial architecture and implementation planning are underway.

## Core Stack
- React + TypeScript + Vite + Tailwind CSS
- Node.js + TypeScript backend
- Supabase
- AI agent orchestration with tool calling

## Important Safety Principle
DocHunter must not provide medical diagnosis.
It should support healthcare navigation and booking assistance only.
