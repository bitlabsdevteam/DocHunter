# DocHunter - AI Agentic Healthcare Locator

Initial scaffold for a Vercel-ready AI healthcare locator focused on Japan.

## Structure
- `app/frontend`: React + Vite client (chat UX, localization-ready)
- `backend`: shared domain and orchestration modules
- `api`: Vercel serverless entrypoints

## Quick start
```bash
npm install
npm run dev
```

## Environment
Copy `.env.example` to `.env` and fill required keys.
